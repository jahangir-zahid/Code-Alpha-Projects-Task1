# 📊 Web Scraping Project – Quotes Dataset

## 📌 Project Overview

This project demonstrates **web scraping using Python** to extract data from a public website and convert it into a structured dataset. The data collected includes quotes and their respective authors.

---

## 🎯 Aim

To extract useful information from a website and store it in a structured format (CSV file) for analysis.

---

## 🛠️ Tools & Technologies Used

* Python
* BeautifulSoup
* requests library
* pandas

---

## 🌐 Data Source

Website used: http://quotes.toscrape.com
(This is a practice website designed for learning web scraping.)

---

## ⚙️ Methodology

1. **Send HTTP Request**
   Used the `requests` library to access the webpage.

2. **Parse HTML Content**
   Used BeautifulSoup to read and navigate HTML structure.

3. **Extract Data**
   Collected:

   * Quote text
   * Author name

4. **Store Data**
   Saved the extracted data into a CSV file using pandas.

---

## 📊 Output

* A file named **quotes.csv** is generated
* Contains:

  * Quote
  * Author

---

## ✅ Results

Successfully extracted data from the website and created a structured dataset in CSV format.

---

## 📚 Conclusion

Web scraping is a powerful technique to collect data from websites. This project shows how Python and BeautifulSoup can be used to gather and store data efficiently for analysis.
